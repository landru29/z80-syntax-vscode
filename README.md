# z80 README

Syntax highlighting for Z80 assembly.

## Features

The Z80 is an 8-bit microprocessor introduced by Zilog.

This extension is the Z80 assembly syntax highlighting.

## Requirements

* Roudoudou assembler: https://github.com/EdouardBERGE/rasm
